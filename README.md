# HEX Tool

A simple Python-based terminal framework with multiple utilities and tools.

---

## ⚙️ Requirements

Before installing and running this tool, make sure you have:

* **Python installed (required)**
* Minimum version: **Python 3.8**
* Recommended version: **Python 3.11**
* Not recommended: **Python 3.13 or newer** (may cause compatibility issues with some libraries)

Make sure Python is added to your **PATH** during installation.

---

## 📦 Installation

To install all required dependencies, run:

```bash
install.bat
```

This will automatically install all required Python packages.

---

## 🚀 Usage

You can start the tool in two ways:

### Option 1 (Recommended)

Run the start file:

```
start.bat
```

---

### Option 2 (Manual start)

If you prefer running it manually:

```
python main.py
```

---

## 📁 Features

* Discord tools
* IP / DNS lookup utilities
* Phone lookup
* Whois lookup
* QR code generator
* System information tool
* Web utilities and more

---

## ⚠️ Notes
